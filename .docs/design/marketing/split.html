01

A real layout engine

True typesetting and composition — not slides with textboxes. Every block (text, image, chart, table, stat, quote, diagram, callout) is a first-class element that re-typesets to fit.

02

One source, three views

Deck, doc, and site render from a single canonical artifact. Switch views instantly, edit once, and every format stays in sync.

36 designer themes

Each is a complete system: a font trio, color, corner radius, border weight, and shadow character. One click restyles the whole artifact, structurally — not just recolored.
